# Cloud project

게시판 회원을 구현하면서 회원별 Cloud system을 구현하기 위해서 만든 웹페이지입니다.

추가적인 설명은 설명 폴더에 있습니다.

메인 페이지는 board.php이며 XMAPP를 활용해 Apache와 MySQL을 활용하여 만들었고 데이터 베이스는 pref.sql를 불러오시면 됩니다.
